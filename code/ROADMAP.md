# 股票基金智能分析系统 - 开发路线图

## 愿景
成为一个真正有用的投资辅助工具，像一个专业投资顾问一样，持续学习、追踪、优化。

## 投资理念
- **价值投资为主**：关注低估值、高股息、稳健增长
- **趋势为辅**：不追涨杀跌，但避免逆势
- **长期持有**：基金定投为主，减少交易
- **风险第一**：保住本金，控制回撤

## 待开发功能 (按优先级)

### P0 - 核心功能
1. **真实数据接口**
   - 修复东方财富API（考虑多个备用源）
   - 添加腾讯财经、网易财经作为备用
   - 港股数据接口（阿斯达克）

2. **历史百分位分析**
   - 计算PE/PB在过去3年/5年的百分位
   - 股价在均线系统的位置
   - 可视化历史估值区间

3. **基金深度分析**
   - 基金经理评分（从业年限、历史管理基金业绩）
   - 持仓变化追踪（季报披露 vs 上季）
   - 跟踪误差分析

### P1 - 重要功能
4. **风险指标**
   - 波动率计算
   - 最大回撤追踪

5. **持仓管理系统**
   - 自动记录买入/卖出

   - 盈亏追踪

6. **提醒系统**
   - 价格突破提醒

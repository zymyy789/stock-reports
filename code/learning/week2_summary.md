## Week2核心收获

### 1. 银行资产负债表分析
- 不良贷款率 <1.5% = 优秀
- 拨备覆盖率 >150% = 安全
- 净息差 = 盈利能力指标

### 2. 保险利润表分析
- NBV增速 = 核心成长指标
- EV = 内含价值
- ROE >12% = 良好

### 3. 杜邦分析
- ROE = 净利率 × 资产周转率 × 杠杆倍数
- 招行ROE13.44% vs 兴业9.15%
- 主要差距在净利率

### 4. 财报异常识别
- 应收增速 > 营收增速 = 危险
- 现金流 < 净利润 = 可疑
- 高管减持 = 强烈预警

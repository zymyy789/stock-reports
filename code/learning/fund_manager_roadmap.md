## 二、学习计划（12周）

### 第1-2周: 投资理念深化
- 价值投资原典精读（格雷厄姆/巴菲特/费雪）
- 成长股投资（彼得林奇/费雪）
- 逆向投资（邓普顿/索罗斯）
- 当前: L3 → 目标: L4

### 第3-4周: 宏观研究与市场周期
- 宏观经济指标体系（GDP/CPI/PMI/利率）
- 货币政策传导机制
- 信贷周期与信用利差
- 美林时钟与资产轮动
- 当前: L2 → 目标: L3+

### 第5-6周: 行为金融学
- 认知偏差（过度自信/锚定/羊群）
- 框架效应与决策陷阱
- 市场情绪量化指标
- 当前: L2 → 目标: L3+

### 第7-8周: 量化分析基础
- Python数据分析（pandas/numpy）
- 统计回归与相关性分析
- 简单回测框架
- 当前: L1 → 目标: L3

### 第9-10周: 交易执行与风控
- 交易成本分析（滑点/佣金/冲击成本）
- 仓位管理系统
- 止损体系设计
- VaR与压力测试
- 当前: L1 → 目标: L3

### 第11-12周: 组合构建与绩效
- 多因子模型
- 风险平价/等权/最大夏普组合
- 绩效归因（Brinson模型）
- 当前: L1 → 目标: L3

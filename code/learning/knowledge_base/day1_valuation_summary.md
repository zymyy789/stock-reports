## 一、DCF现金流折现

### 核心原理
- 今天的钱比明天的钱值钱
- PV = CF1/(1+r)^1 + CF2/(1+r)^2 + ... + CFn/(1+r)^n

### 4大假设（最容易出错）
1. 折现率(r): 10%还是12%？差2%导致结果差30%+
2. 增长率(g): 3%还是5%？差2%导致结果差60%+
3. 现金流预测: 没人能准确预测10年后
4. 折现年数: 通常10年，但有些公司需更久

### 适用场景
- ✅ 现金流稳定可预测（公用事业、消费龙头）
- ❌ 银行/券商/保险（现金流难定义）
- ❌ 周期股（利润波动大）
- ❌ 亏损公司（没有现金流）

### 关键教训
- 单一DCF数字没有意义，必须看敏感性区间
- DCF是思考框架，不是精确数字
- 银行估值用PB/PE/股息率，不用DCF

## 二、PE/PB估值

### PE的5种类型
1. 静态PE(LYR): 去年EPS，滞后
2. 动态PE(TTM): 过去12月EPS，受一次性损益影响
3. 预期PE(Forward): 分析师预测，可能不准
4. PEG: PE/增长率，彼得林奇发明
5. CAPE: 10年平均通胀调整，席勒发明

### PB的关键理解
- PB < 1 = 破净 = 市场认为资产可能贬值
- PB > 1 = 溢价 = 市场认为有商誉/增长
- 长期PB会回归1

### PE/PB/ROE三角关系
- PE = PB / ROE
- 隐含ROE = PB / PE × 100%
- 如果隐含ROE ≈ 实际ROE → 估值合理

### 低PE陷阱
- PE < 5 不一定是机会，可能是：
  - 周期股顶点（利润最好的时候）
  - 夕阳行业
  - 财务造假
  - 即将倒闭

## 三、历史百分位

### 判断标准
- < 20%: 极度低估 → 强烈买入
- 20-40%: 低估 → 买入
- 40-60%: 合理 → 持有
- 60-80%: 高估 → 谨慎
- > 80%: 极度高估 → 卖出

### 为什么适合A股
- 散户为主，情绪波动大
- 错误定价更常见
- 均值回归概率高

### 局限性
- 时间窗口选择影响结果
- 均值回归不一定发生（行业衰退）
- 忽略基本面变化
- 极端情况可能继续走极端

### 实战技巧
- 多周期验证（5年/10年/15年）
- 结合绝对PE值
- PE百分位低 + PB百分位低 = 双重低估
- 分批建仓

## 四、行业差异化估值

### 估值方法选择决策树
| 行业 | 核心资产 | 最佳估值方法 |
|------|----------|-------------|
| 银行 | 贷款 | PB + 股息率 |
| 保险 | 保单 | PEV + NBV |
| 券商 | 客户 | PB + 市场情绪 |
| 白酒 | 品牌 | PE + 品牌溢价 |
| 消费 | 品牌+渠道 | PE + PEG |
| 科技 | 专利+人才 | PEG + PS |
| 公用事业 | 特许经营 | DCF + 股息率 |
| 周期股 | 产能 | PB + 周期位置 |

### 持仓验证
- 兴业: PB=0.49 + 股息3.14% → 极度低估
- 招行: PB=0.89 + 股息7.67% → 低估
- 平安: PEV=0.7 + NBV+29% → 极度低估
- 中信: PB=1.0 + 券商周期底部 → 合理
- 伊利: PE=20.6 + 毛利率30% → 合理持有
- 五粮液: PE=14 + 毛利率70% → 低估

## 五、银行螺丝钉理念

### 核心方法
1. 低估定投：PE/PB百分位<30%时买入
2. 指数基金为主：降低个股风险
3. 定期定额：摊低成本
4. 高估卖出：百分位>70%时卖出

### 与我的策略对比
- 我：主动选股 + 一次性买入 + 5-12只
- 螺丝钉：指数基金 + 定投 + 长期持有
- 改进：可加入定投策略 + 指数基金底仓

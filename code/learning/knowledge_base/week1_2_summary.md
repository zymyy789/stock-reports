## 一、格雷厄姆 - 价值投资之父

### 三大核心
1. **安全边际**: 买入价 < 内在价值，至少打折20-30%
2. **市场先生**: 接受他的报价，但自己判断
3. **防御/积极型**: 防御型=债5股5分散；积极型=精选个股

### 格雷厄姆选股七条
1. 公司规模不能太小
2. 财务稳健 (流动比率>2)
3. 连续20年支付股息
4. EPS连续20年增长
5. PE < 15
6. PB < 1.5
7. 股价 < 净资产的2/3

### 我的应用
- 兴业PB=0.49 ✅
- 招行PE=6.6 ✅

## 二、巴菲特 - 进化的价值投资

### 三大核心
1. **买股票=买公司**: 用做生意的心态，不看股价
2. **能力圈**: 不懂不买，只懂银行/保险/消费
3. **时间是好公司的朋友**: ROE13%，10年复合增长

### 巴菲特风险观
- 普通人: 风险=股价下跌
- 巴菲特: 风险=本金永久损失
- 短期波动不是风险，买贵了才是

### 护城河
- 品牌/专利/网络效应/低成本
- 招行零售银行护城河 ✅
- 平安综合金融护城河 ✅

## 三、彼得林奇 - 成长股投资

### 股票六分法
1. 缓慢增长型 (公用事业/大银行) - 拿股息
2. 稳定增长型 (招行/五粮液) - 低估买高估卖
3. 快速增长型 (早期苹果) - 高增长，小变大
4. **周期型** (银行/券商) ← 我的持仓
5. 资产型 (土地/品牌)
6. 困境反转型

### PEG估值
- PEG = PE / 增长率
- PEG < 0.75 极度低估
- PEG < 1 低估
- PEG > 2 危险

### 我的持仓PEG
- 招行: PEG=1.1 (银行不适合PEG)
- 平安: PEG=0.27 ✅ 极度低估
- 伊利: PEG=2.6 ❌ 偏高

## 四、邓普顿 - 逆向投资

### 核心
- "牛市生于悲观，长于怀疑，死于乐观"
- "街头溅血时买入"
- 所有人都嫌弃时 = 最佳买入时机

### 邓普顿估值
预期回报 = (盈利增长 + 股息) / PE

- 招行: (6%+7.67%)/6.6 = 2.07% ✅
- 兴业: (5%+3.14%)/5.1 = 1.60% ✅
- 平安: (6.5%+2%)/7.9 = 1.08% ✅

## 五、综合：我的投资哲学

```
以合理价格买低估的好公司，
在所有人悲观时买入，
长期持有，直到高估或基本面破坏。
```

### 操作标准
- **买入**: PE<10 or PB<0.8 + 股息>3%
- **持有**: 3年内不因短期波动卖出
- **卖出**: PE>25 or 基本面破坏 or 发现更好的

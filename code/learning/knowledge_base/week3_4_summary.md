## 二、货币政策传导

降息对持仓的影响:
- **银行**: 短期利好(预期改善)，长期净息差收窄
- **保险**: 复杂，短期利好>利空
- **消费**: 利好(消费意愿提升)

5. **确认偏误**: 只看利好 → 定期找反对意见

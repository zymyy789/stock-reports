## 投资体系5步法

### Step 1: 选股
- PE<15 + ROE>10 + 负债率合理
- 银行用PB, 保险用PEV, 消费用PE

### Step 2: 估值
- 历史百分位<30%关注
- 安全边际至少20%
- 多指标交叉验证

### Step 3: 买入
- 5-10只, 3+行业
- 单只<20%, 行业<40%
- 分批买入, 留10-20%现金

### Step 4: 持有
- 不看短期波动
- 定期检查基本面
- 每月复盘

### Step 5: 卖出
- PE百分位>80%
- 基本面恶化
- 发现更好标的
- 逻辑被破坏

---

## 5个最重要认知

1. 估值方法必须匹配行业
2. 安全边际是基石
3. 单一指标不可靠
4. 现金流>利润
5. 投资是认知变现

# Week1 学习成果总结

## 核心知识

### 1. DCF估值局限性
- 参数敏感：增长/折现率微调→结果大变
- 需要准确预测10年现金流
- 结论：DCF作参考，结合机构观点更可靠
- 参考：机构预测往往更准确

### 2. 历史百分位分析
- 银行螺丝钉规则：
  - PE/PB < 30%分位 → 买入
  - 30-70%分位 → 持有
  - > 70%分位 → 卖出
- 银行股普遍在10-15%分位 → 极度低估

### 3. PEG深入理解
- 公式：PEG = PE / 净利润增长率
- PEG < 1 → 成长被低估
- PEG > 2 → 成长被高估
- 注意：周期股/金融股需要调整

### 4. 行业差异化估值
| 行业 | 核心指标 | 估值方法 |
|------|----------|----------|
| 银行 | PB + 股息率 | PB<1破净=低估 |
| 保险 | NBV增速 + EV | NBV增长29%=优秀 |
| 券商 | PEG + 政策 | 周期股，政策敏感 |
| 消费 | PE + 品牌 | ROE>15%=优质 |

### 5. 为什么同一PE但估值不同
- 中信PE=11.9, 增长38.6% → PEG=0.31 → 极度低估
- 平安PE=7.9, 增长6.5% → PEG=1.22 → 估值合理
- 保险业看NBV而非PE
